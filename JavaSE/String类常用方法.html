String类常用方法

JavaDoc：Java的API文档

字符串和字符数组：

构造函数 public String（Char[] value）将全部字符数组变成字符串

构造函数 public String（Char[] value，int offset ,int count）将部分字符数组变成字符串

普通方法 public  char charAt( int offset)   获取指定索引位置的字符

普通方法  public  char[] toCharArray()   将字符串转换为字符数组返回

字符串和字节转换：

构造函数 public String（byte[] bytes）将全部字节数组变成字符串

构造函数 public String（ byte[] bytes ，int offset ,int count）将部分字节数组变成字符串  

普通方法 public byte[] getBytes()将字符串转换为字节数组

普通方法 public byte[] getBytes(String charsetName) throws UnsupportedEncodiingException  编码转换


字符串比较：

普通方法 public boolean equals（String object）大小写是区分的字符比较

普通方法  public boolean equalsIgnoreCase（String anotherString）大小写不区分的 字符表叫

普通方法   public  int compareTo（ String anotherString ） 大小写是区分的 进行字符串大小比较：int返回值可以<0;   0<;   0=;

普通方法   public  int compareTo IgnoreCase （ String str） 大小写不区分的  进行字符串大小比较

字符串查找功能：

普通方法 public boolean contains​(String s) 判断子字符串是否存在

普通方法 public int indexOf​(String str) 从前向后查找指定字符串的位置

普通方法  public int indexOf​(String str, int fromIndex)  从指定位置 由前向后 查找字符串的位置

普通方法  public int lastIndexOf​(String str)    从后往前查找字符串的位置

普通方法  public int lastIndexOf​(String str, int fromIndex) 从 指定  位置后往前查找字符串的位置 

普通方法  public boolean startsWith​(String prefix) 判断是否以指定字符串开头

普通方法  public boolean startsWith​(String prefix, int toffset) 判断是否以指定位置指定字符串开头

普通方法 public boolean endsWith​(String suffix) 判断是否以指定字符串开结尾

字符串替换：

  普通方法 public String replaceAll​(String regex, String replacement)  全部替换

普通方法 public String replaceFirst​(String regex, String replacement)   替换 首个

牵扯到正则表达式

字符串拆分：字符串拆分成字符串数组的形式返回

普通方法  public String[] split​(String regex) 按照指定的字符串全部拆分

普通方法  public String[] split​(String regex, int limit) 按照指定的字符串拆分指定的个数，后面不再拆分  

部分拆分需要\\转义才能拆分

字符串的截取：

普通方法   public String substring​(int beginIndex) 从指定索引开始截取

普通方法   public String substring​(int beginIndex, int endIndex) 从指定范围做截取

开发中常常使用indexOf()计算得来的

格式化字符串：类似C语言占位符输出%s,    %d,   %f,   %c

普通方法  public static String format​(String format, Object... args) 根据指定的结构进行文本输出

字符串其他方法：

普通方法  public String concat​(String str) 字符串连接按照处理，没有按照静态的处理

普通方法 public String intern​() 对字符串对象入池保存

普通方法  public boolean isEmpty​() 判断内容是否是空字符串

普通方法   public int length​() 判断字符串长度

普通方法   public String trim​() 去除左右的空格数间歇

普通方法  public String toUpperCase​() 转大写，避免非字母 转换

普通方法 public String toLowerCase​() 转小写 ，避免非字母 转换
