包装类

Java中的基本数据类型没有方法和属性，而包装类就是为了让这些拥有方法和属性，实现对象化交互。   

数值型包装类都继承至Number。

字符型和布尔型继承至Object。 




基本数据和包装类之间的转换

装箱：基本数据类型转换为包装类； 

拆箱：包装类转换为基本数据类型。

基本数据类型和包装类的转换

通过包装类Integer.toString()将整型转换为字符串；

通过Integer.parseInt()将字符串转换为int类型；

通过valueOf()方法把字符串转换为包装类然后通过自动拆箱。

Number类是个抽象类：

普通方法  public byte byteValue​() 从包装类中获取Byte数据

普通方法  public short shortValue​() 从包装类中获取short数据

普通方法  public abstract double doubleValue​()  从包装类中获取 double 数据

普通方法  public abstract float floatValue​() 从包装类中获取 float 数据

普通方法  public abstract long longValue​() 从包装类中获取 long 数据

普通方法  public abstract int intValue​() 从包装类中获取 int 数据

装箱和拆箱：

数据装箱：将基本数据保存到包装类中，通常通过构造方法实现。

|-Integer类： public  Integer （int valve）； 

|- Double类： public   Double （ double valve）； 

|- Boolean类： public    Boolean  （  boolean  valve）；

数据拆箱：从包装类中获取基本类型：

|-数据类型包装类已经由Number类定义了拆箱的方法了；

|-Boolean型： public boolean booleanValue​()；

自动装箱可以支持自动的数学运算

Integer  obj=10；//自动装箱

obj++；

int  num=obj；//自动拆箱

还可以实现Object接收基本数据类型的操作

Object obj=19.2；//double自动装箱为Double,向上转型Object类

double num=（Double） obj；//向下转型成包装类才能拆箱

包装类的相等判断需要equals去判断；
