This指针

使用this可以实现三类结构描述：

当前类的属性，this属性

当前类中的方法（普通方法，构造方法）：this（），this。方法名称

描述当前对象

使用this指针调用当前类中属性

public person（String name ,int age）{

                            name=name;

                              age=age

}

这样导致类中的name ,age 不能与参数 name , age 所分辨。导致对象数据无变化正确写法如下：

public person（String name ,int age）{   

                                     this.name=name;                          

                                     this.age=age

}    

this调用本类方法：

普通方法调用问题不大，可以加上this，也可不加

对于构造方法，利用this调用本类的构造方法；

对本类的构造方法自我调用：

this语句只可以被放在构造函数 的第一行，构造方法可以调用普通方法，普通方法不可以调用构造方法
